Formatações

Negrito ou Destaque

Com as formatações podemos usar formas como Negrito não semantico usando a tag <b>.Mas podemos tambem fazer o Negrito da maneira semantica usando a tag Strong

Italico ou Enfase

Da mesma forma podemos em Italico da forma nao semantica usando a tag <I>.E assim podemos criar um Itaico semantico usando a tag <em>.

Marcação

Para criarmos um texto marcado de forma semantica utilizamos a tag <mark>

Tamanhos de Texto

Criaremos um texto grande (não semantica) com a tag <big>, e um texto pequeno com a tag <small>.

Textos Excluidos

Podemos tambem criar textos excluidos usando a tag <del>.

Texto Inseridos

Podemos criar Textos Inseridos com a tag лins>.

Texto Sobrescrito

Para criar coisas sobrescritas como xpo+7 utilizamos a tag <sup>

E por fim podemos criar coisas como H7O.
